## Select

The entire icon set from React Select is approved for use.

Start by installing <code>react-select</code>
<code>yarn add react-select</code>

Import the default export and render in your component:
<code>import React, &#123; Component &#125; from 'react';
<p>import Select from 'react-select';</p>
  const options = [<br />
  &ensp;&#123; value: 'chocolate', label: 'Chocolate' &#125;,<br />
  &ensp;&#123; value: 'strawberry', label: 'Strawberry' &#125;,<br />
  &ensp;&#123; value: 'vanilla', label: 'Vanilla' &#125;<br />
  ]<br />

  const MyComponent = () => (
    &#60;Select options=&#123;options} /&#62;
  )
</code>

Full information you can read on official documentation: <a href="https://react-select.com/home" target="_blank" rel="noreferrer">react select docs</a>

The select component allows users to choose single or multiple options from a list.
