# Toggle Button Group
To emphasize groups of related Toggle buttons, a group should share a common container.
The ```ToggleButtonGroup``` controls the selected state of its child buttons when given its own ```value``` prop.

## Style

### Sizes
Supports two sizes: ```small``` and  ```large```.

<Canvas>
  <Story name="Small" story={stories.Small} />
  <Story name="Large" story={stories.Large} />
</Canvas>

### Types
Buttons can also be used to style ```checkbox``` and ```radio``` form elements.
This is helpful when you want a toggle button that works neatly inside an HTML form.
