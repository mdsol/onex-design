#### Body
The component renders a ```<p>``` element.
With variant ```body``` also used argument: ```href```.
Providing a ```href``` will render an ```<a>``` element.
Providing a ```semiBold```  argument will render an ```<p>``` element with width ```600```.

  <Story name="Body" story={stories.Body} />

</Canvas>

#### Label
The component renders a ```<span>``` element.
With variant ```label``` also used argument: ```href```.
Providing a ```href``` will render an ```<a>``` element.
Providing a ```semiBold```  argument will render an ```<span>``` element with width ```600```.

  <Story name="Label" story={stories.Label} />

#### Caption
The component renders a ```<span>``` element.
With variant ```caption``` also used arguments: ```uppercase``` and ```href```.
Providing a ```href``` will render an ```<a>``` element.
Providing a ```semiBold```  argument will render an ```<span>``` element with width ```600```.
Providing a ```uppercase```  argument will render an ```<span>``` element with text-transform ```uppercase```.
