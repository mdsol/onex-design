Hint informs the user about important information related to an action.

### Hint variants

The component has 4 variants such as "info", "success", "warning", "error".

##### <i>info</i>

Information hint helps users to understand the context of interacting with the action.

<Canvas>
  <Story name="Info" story={stories.Info} />
</Canvas>

##### <i>success</i>

Success hint gives the user local feedback about successful interaction with the system.

  <Story name="Success" story={stories.Success} />

##### <i>warning</i>

Warning hint informs the user of non-critical issues or blockers.
