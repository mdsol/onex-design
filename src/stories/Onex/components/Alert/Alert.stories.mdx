# Alert

Provide contextual feedback messages for typical user actions with the handful of available and flexible alert messages.

## Examples
Alerts are available for any length of text, as well as an optional dismiss button.For proper styling, use one of the eight variants.

<Canvas>
  <Story name="Examples" story={stories.Examples}/>
</Canvas>

## Actions

For Actions, use the```primaryAction``` and```secondaryAction``` props to provide matching colored Actions within any alert. It's requered using ```body``` prop before, in other case it not will be visible.

<Canvas>
  <Story name="Actions" story={stories.Actions}/>
</Canvas>

## Additional content

Alerts can contain whatever content you like.Headers, paragraphs, dividers, go crazy. Use ```body``` prop for that.

  <Story name="AdditionalContent" story={stories.AdditionalContent}/>
</Canvas>
