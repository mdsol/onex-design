## Controlled
You can control the switch with the ```checked``` and ```onChange``` props.
<Canvas>
  <Story name="Controlled">
    {() => {
      const [checked, setChecked] = React.useState(true);
      return (
        <>
          <Switch
                  checked={checked}
                  onChange={() => setChecked(!checked)}
                  id="custom-switch-checked"/>
        </>
      );
    }}
  </Story>
</Canvas>
