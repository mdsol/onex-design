# Stacks

Shorthand helpers that build on top of our flexbox utilities to make component layout faster and easier than ever.

## Vertical
Stacks are vertical by default and stacked items are full-width by default. Use the gap prop to add space between items.

<Canvas>
    <Story name="Vertical" story={stories.Vertical} />
</Canvas>

## Horizontal

Use ```direction="horizontal"``` for horizontal layouts. Stacked items are vertically centered by default
and only take up their necessary width. Use the ```gap``` prop to add space between items.

<Canvas>
    <Story name="Horizontal" story={stories.Horizontal} />
</Canvas>
