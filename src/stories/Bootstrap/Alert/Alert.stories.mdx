# Alert

Provide contextual feedback messages for typical user actions with the handful of available and flexible alert messages.

## Examples
Alerts are available for any length of text, as well as an optional dismiss button.
For proper styling, use one of the eight variants.

<Canvas>
  <Story id='bootstrap-components-alert--examples' />
</Canvas>

## Links

For links, use the ```<Alert.Link>``` component to provide matching colored links within any alert.

<Canvas>
  <Story id='bootstrap-components-alert--links' />
</Canvas>

## Additional content

Alerts can contain whatever content you like. Headers, paragraphs, dividers, go crazy.

<Story id='bootstrap-components-alert--additional-content' />

## Dismissing
Add the ```dismissible``` prop to add a functioning dismiss button to the Alert.

  <Story id='bootstrap-components-alert--dismissing-error' />
