# Popovers
A popover component, like those found in iOS.

You can control the placement of the Popover.

# Demo
<Canvas>
    <Story name="Demo"
           args={{
               placement: 'right',
               id: 'popover-basic'
           }}>
        {(args) => {
            const { placement } = args;
            return (
                <OverlayTrigger trigger="click" placement={placement} overlay={
                    <Popover {...args}>
                        <Popover.Header as="h3">Popover {placement}</Popover.Header>
                        <Popover.Body>
                            And here's some <strong>amazing</strong> content. It's very engaging.
                            right?
                        </Popover.Body>
                    </Popover>
                }>
                    <Button variant="primary">Click me to see</Button>
                </OverlayTrigger>
            );
        }}
    </Story>
</Canvas>
